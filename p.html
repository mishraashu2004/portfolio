About Me

Crafting a Digital Presence
As a dedicated frontend website developer, I've honed my skills to create visually appealing and user-friendly web experiences. My journey has been enriched by internships that have solidified my understanding of modern web development frameworks.

I've successfully completed internships in MEAN and MERN stack development at Codebit, gaining practical experience in building dynamic web applications. My foundational knowledge in HTML, CSS, and JavaScript has been instrumental in crafting robust and responsive web interfaces.

Beyond web development, I've delved into the realm of mobile app development, mastering Android development using Android Studio and Jetpack Compose. This experience has broadened my skillset and allowed me to create innovative and intuitive mobile applications.

To enhance user authentication and data management, I've acquired expertise in Google Firebase Authentication. This knowledge has proven invaluable in building secure and scalable web and mobile applications.

In addition to my frontend and mobile development skills, I have a strong foundation in general-purpose programming languages like Python, C, and Java. This diverse skill set enables me to approach problem-solving from multiple angles and contribute effectively to various development projects.

I'm passionate about staying updated with the latest trends and technologies in the web development industry. My goal is to continuously expand my knowledge and deliver high-quality, innovative solutions that exceed client expectations.
